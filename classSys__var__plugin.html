Detailed Description

The class for variables which value is a plugin. Example: @default_storage_engine

Class specific constructor arguments: int plugin_type_arg (for example MYSQL_STORAGE_ENGINE_PLUGIN)

Backing store: plugin_ref

Note:
these variables don't support command-line equivalents, any such command-line options should be added manually to my_long_options in mysqld.cc
