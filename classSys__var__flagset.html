Detailed Description

The class for flagset variables - a variant of SET that allows in-place editing (turning on/off individual bits). String representations looks like a "flag=val,flag=val,...". Example: @optimizer_switch

Class specific constructor arguments: char* values[] - 0-terminated list of strings of valid values

Backing store: ulonglong

Note:
the last value in the values[] array should always* be the string "default".
